2. Main

The Org-mode manual indicates these possible values in initial visibility: 

#+startup: overview
#+startup: content
#+startup: showall
#+startup: showeverything

With the following meanings, according to the summary of in-buffer settings:

overview top-level headlines only content all headlines showall no folding of any entries showeverything show even drawer contents
