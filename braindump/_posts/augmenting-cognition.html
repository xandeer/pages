1. How to sever dependencies?   MostlyCloudy MostlySunny @home

Use mirror types. 

type OriginalType = {
  a: string,
  b: string,
  c: () => string
}

type MirrorType = {
  c: () => string
}

function fn(x: string | MirrorType): string {
  if (typeof x === 'object') {
    return x.c()
  } else {
    return x
  }
}

const o = {
  a: 'a',
  c: () => 'c'
}

fun main() = runBlocking {
  val channel = Channel<Int>()
  launch {
    for (x in 1..5) channel.send(x * x)
    channel.close()
  }

  for (y in channel) println(y)
}
