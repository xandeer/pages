Google Analytics on Android

links
Android
  1. Follow the Start Doc
  2. Support multiple environments, The Google Services Gradle Plugin

  3. Validate debugView: 

       adb shell setprop debug.firebase.analytics.app <package_name>

  4. Track events: 

       fun track(event: String, param: String) {
   val bundle = Bundle()
   bundle.putString("param", param)
   // Firebase doesn't support event name contains space.
   FirebaseAnalytics.getInstance(App.context)
       .logEvent(event, bundle)
   }
  5. Get deobfuscated crash reports with the Firebase Crashlytics SDK

  6. Check if crashlytics is working? 

       throw RuntimeExpection("Test")

    Then, restart app.
