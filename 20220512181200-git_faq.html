1.4. Is clean?

# clean
[ -z "$(git status --porcelain)" ]
# dirty
[ -n "$(git status --porcelain)" ]

git status --porcelain

1.5. How to check out files from another branch?

git switch feature/A
git restore --source feature/B -- a.ts

1.6. How to clone to a specified directory?

git -C dir clone repo

1.7. How to commit with multiline messages?

Set git config core.editor = emacsclient, then git commit without -m will open emacs. 

git commit -m "subject line" -m "content line"

1.8. How to undo a merge?

git reset --merge HEAD~1
