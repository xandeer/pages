6.2. How to tell whether a symbol is in the type space or value space?

  • Every value has a type, but types do not have values. Constructs such as type and interface exist only in the type space.
  • typeof, this and many other operators and keywords have different meanings in type space and value space.
  • Some constructs such as class or enum introduce both a type and a value.

6.3. How to declare something like map?

7. Kotlin Coroutines   Kotlin Coroutines

7.1. A coroutine job is a {{c1::cancellable}} thing with a {{c1::life-cycle}} that culminates in its completion.

7.2. What is the throttle with kotlin coroutine flow?

fun <T> Flow<T>.sample(periodMillis: Long): Flow<T>

7.3. Kotlin coroutine {{c1::async}} return a {{c1::Deferred}} instance, which is a specialized {{c1::Job}} with a return value.

7.4. What is a CoroutineScope in kotlin coroutine?

A CoroutineScope is an object that plays the role of the parent in Structured cocurrency–its purpose is to manage and monitor the coroutines you create inside it.

In other words, a CoroutineScope is a container for a CoroutineContext. 

interface CoroutineScope {
  val coroutineContext: CoroutineContext
}

7.5. What is a CoroutineDispatcher in kotlin coroutine?

A CoroutineContext used to specify the thread scheduler.

7.6. What is a Job in kotlin coroutine?

A CoroutineContext used to save coroutine's life-cycle.

7.7. What is a SupervisorJob in kotlin coroutine?

It's a special Job, it's children can fail independently of each other.

7.8. How to handle uncaught exceptions in a launch in kotlin coroutine?

Set a CoroutineExceptionHandler to CoroutineScope: 

val ceh = CoroutineExceptionHandler { _, exception -> println("Caught $exception") }
val scope = CoroutineScope(coroutineContext + ceh)
scope.launch {
  // ...
}

// or
launch(ceh) {
  // ...
}

7.9. How to handle exceptions in a async in kotlin coroutine?

Handle it when call Deferred.await: 

val a = async {
  delay(100)
  throw RuntimeException("Error")
  42
}

try {
  a.await()
} catch (e: RuntimeException) {
  println("Caught $e")
}
