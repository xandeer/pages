ParcelFileDescriptor#parseMode(java.lang.String):

"t" indicates the file should be opened in truncate mode, equivalent to OsConstants#OTRUNC. If the file already exists and is a regular file and is opened for writing, it will be truncated to length 0.

Without "t", maybe something strange will happen. 

context.contentResolver.openOutputStream(uri, "wt")
