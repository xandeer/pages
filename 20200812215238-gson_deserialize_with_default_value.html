data class 反序列化时，如果字符串中有 null ，设置的默认值会被覆盖掉。 

data class A(
  val b: Boolean,
  val c: String = ""
)

解决方案：将 data class 换成普通 class ，并将构造参数全部移除。如下： 

class A {
  val b: Boolean = true
  val c: String = ""
}
