1 Install

brew install clojure/tools/clojure

wget https://raw.githubusercontent.com/technomancy/leiningen/stable/bin/lein -O ~/.bin
chmod +x ~/.bin/lein

2 Start

take ~/temp/lein
lein new app hello
emacsclient hello/project.clj

(cider-jack-in
 '(:project-dir "~/temp/lein/hello"))
