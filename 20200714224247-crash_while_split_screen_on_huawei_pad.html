Crash while Split Screen on Huawei Pad

links
Donut Bug Fixed

The problem is about that split screen will dispatch onDetachedFromWindow and onAttachedToWindow each time. So don't depend upon them to do initializations. 

override fun onAttachedToWindow() {
  super.onAttachedToWindow()
  // Below will be called as splitting screen.
  addView(et, matchParent, wrapContent)
  subscribeVms()
}
